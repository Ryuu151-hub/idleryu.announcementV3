# IDLERYU Announcement Repository

Remote announcement server for the Chrome Extension.

## Files

announcement.json

Contains:

- Announcement
- Optional update
- Force update
- Changelog
- CTA button

version.json

Contains:

- Latest version
- Minimum version
- Maintenance mode
- Blacklisted versions

Deploy to Vercel.

Example URL:

https://your-project.vercel.app/announcement.json

https://your-project.vercel.app/version.json
